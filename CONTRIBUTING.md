# Contributing to Project Restore Hope

Thank you for your interest in contributing to Project Restore Hope's website. This is a nonprofit organization serving children and families in Mukuru, Kenya. Every contribution helps us extend our reach and impact.

## Code of Conduct

By participating, you agree to maintain a respectful, inclusive, and constructive environment. Harassment or discriminatory behavior of any kind will not be tolerated.

## How to Contribute

### Reporting Issues

- **Bug reports** — Open an issue with a clear description, steps to reproduce, expected vs. actual behavior, and screenshots if applicable.
- **Feature requests** — Open an issue describing the feature, why it would be valuable, and any implementation ideas.
- **Content corrections** — If you spot inaccurate or outdated information about the organization or its programs, please report it.

### Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes following the guidelines below
4. Test your changes across multiple pages
5. Commit with a clear message (`git commit -m "Add feature: your feature"`)
6. Push to your fork (`git push origin feature/your-feature`)
7. Open a Pull Request against the `main` branch

### Development Guidelines

- **Do not change** any content, branding, images, colors, mission statements, or nonprofit messaging without explicit approval.
- **Preserve** all existing class names, IDs, and data attributes that the JavaScript depends on.
- **Maintain** backward compatibility — do not break existing functionality.
- **Follow** the existing code style and patterns.
- **Test** on mobile, tablet, and desktop viewports.
- **Verify** accessibility with keyboard navigation and screen readers.

### Code Style

- HTML: Use semantic HTML5 elements (`<main>`, `<section>`, `<nav>`, `<article>`)
- CSS: Use CSS custom properties from `:root` for colors, fonts, and spacing
- JavaScript: Use vanilla JS (no frameworks), `const`/`let`, descriptive variable names
- Indentation: 2 spaces

## Areas Needing Help

- Image optimization (convert JPEGs to WebP, compress large files)
- Accessibility improvements
- Performance optimization
- Content updates (news, student profiles)
- Translation / internationalization
- Testing across browsers and devices

## Questions?

Open an issue or contact the maintainers directly.
